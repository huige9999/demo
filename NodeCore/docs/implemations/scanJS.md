# 扫描目标目录下的js文件


## 思路
1. 通过Node.js的`fs`模块读取目标目录下的所有文件和子目录。
2. 递归地遍历子目录，继续读取其中的文件和子目录。
3. 对于每个文件，检查其扩展名是否为`.js`，如果是，则将其路径添加到结果列表中。
4. 最终返回所有js文件的路径列表。


## 相关API
- `fs.readdirSync(path)`: 同步读取目录内容，返回一个包含目录中所有文件和子目录名称的数组。
- `fs.statSync(path)`: 同步获取文件或目录的状态信息，返回一个包含文件或目录属性的对象。
- `path.extname(path)`: 获取文件的扩展名，返回一个字符串。
- `path.join(...paths)`: 将多个路径片段连接成一个路径字符串，并正确处理路径分隔符。
